## Running the app

```bash
# development
$ yarn run dev

# build & run in production mode
$ yarn run prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
