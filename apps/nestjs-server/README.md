### Features
- Routes directory
- Code snippet to switch to `platform-express` or `platform-fastify` (remove the unused platform from package.json)

## Running the app

```bash
# development
$ pnpm run dev

# build & run in production mode
$ pnpm run prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
